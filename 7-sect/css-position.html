CSS Positioning

  • Value of positioning

    • static - default
    • relative - position relative to default position
    • absolute - position relative to nearest positioned ancestor or top left corner of webpage
      [ancestor position: relative; and descendent position: absolute;]
      if no ancestor positioned then part 2 comes
    • fixed - position relative to top left corner of browser window (used in navbar)

  • Z-index

    • which element goes on top of which
    • default z-index: 0
    • z-index: 1 to come on top
    • z-index: -1 to go behind
