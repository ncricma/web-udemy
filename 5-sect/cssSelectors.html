CSS Selectors

  • Element selector

    • like h1, p, ul

  • Class selector

    • starts with a . dot, like .red-heading, .container

    many elements can have it

  • Id selectors

    • starts with #, like #main, #blue

    unique element will have it

  • Attribute selector

    • eg p[draggable]{ color: red; }

  • Universal selector

    • *{color: red;} means select all
