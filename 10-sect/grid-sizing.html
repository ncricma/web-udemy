Grid Sizing

display: grid;

gird-template-rows: 100px auto; (auto is used to take space in which the content gets fit)

gird-template-columns: 200px auto; (auto will take the remaining space of the screen)

Example

In chessboard example:

we can replace gird-template repeated code with:

gird-template-rows: repeat(2, 200px)

gird-template-columns: repeat(2, 100px)
